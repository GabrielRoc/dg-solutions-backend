## Descrição

Implementação de um back-end para uma lista de pessoas com nome, data de nascimento e idade(gerada)

## Instalação

```bash
# setting up environment variables file
$ cp .env.example .env

# installing dependencies
$ yarn
```

## Execução

```bash
# development
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Testes

```bash
# e2e tests
$ yarn test:e2e
```
